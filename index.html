josh.ai Code Challenge Web App

Getting Started:

  1. Open your browser's Dev Tools to observe the JS runtime console for this app
  2. In a terminal, start the Hue Bridge Simulator serving to localhost:80
  3. In separate browser window/tab, browse to Hue Bridge Simulator at http://localhost:80
  4. Click the [Link-Button] button (to put the bridge into "link" mode)
  5. Within 30 seconds, refresh the josh.ai app window/tab to connect it to the bridge in privileged mode
  6. Monitor browser Dev Tools console for messages
  7. Expect: initial light status reported
  8. Expect: light status changes to be reported after setting (changing) state of light in bridge simulator manually.
  9. Note: Application continuously polls bridge every 2 seconds for changes, and will stop polling after 60 continuous seconds without any detected light changes.

* If you see "link button not pressed" error message, try clicking [Link-Button] again in bridge simulator window/tab and reloading the app page.

The Jasmine unit test suite (below) will automatically run on page load.
